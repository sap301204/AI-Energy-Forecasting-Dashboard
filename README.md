# ⚡ AI Energy Forecasting Dashboard

An AI-powered web application that predicts energy consumption based on time inputs using Machine Learning and displays insights through an interactive dashboard.

---

## 🚀 Features

* 🔮 Energy Prediction using ML Model
* 📊 Interactive Dashboard with Charts
* ⚡ Real-time API Integration (Flask + React)
* 📈 Hourly & Weekly Load Visualization
* 🤖 AI Insight Panel

---

## 🛠 Tech Stack

**Frontend:**

* React (Vite)
* Axios
* Recharts

**Backend:**

* Flask
* Scikit-learn
* Pandas

---

## ⚙️ Setup Instructions

### 1️⃣ Backend

```bash
cd backend
pip install -r requirements.txt
python app.py
```

---

### 2️⃣ Frontend

```bash
cd frontend/my-app
npm install
npm run dev
```

---

## 🌐 Run Application

* Frontend → http://localhost:5173
* Backend → http://127.0.0.1:5000
